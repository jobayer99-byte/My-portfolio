# 🔥 Cyber Portfolio - React Edition

A stunning neon dark portfolio with advanced animations and effects built with React + Vite.

## ✨ Features

- ✅ Neon Dark UI with cyberpunk aesthetics
- ✅ Skill hover animations
- ✅ Scroll animations (AOS)
- ✅ Hire Me popup modal
- ✅ Platform links
- ✅ Education cards
- ✅ Responsive layout
- 🔥 Typing name animation
- 🔥 3D hacker background
- 🔥 Cyber particle animation
- 🔥 Matrix effect
- 🔥 Skills progress animation
- 🔥 Super smooth scrolling

## 🚀 Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## 🎨 Customization

1. Replace placeholder image in `Hero.jsx`
2. Update your information in all components
3. Add your CV file and update download link in `Navbar.jsx`
4. Customize colors in `src/index.css` (CSS variables)
5. Update social media links in `Hero.jsx`
